REST

This API supports a Representational State Transfer (REST) model for accessing a set of resources through a fixed set of operations. The following resources are accessible through the RESTful model:

  • /group/create
  • /group/list
  • /group/listOwnedGroups
  • /group/listSubscriptions
  • /group/search
  • /message/create
  • /user/register
  • /group/{groupId}/delete
  • /group/{groupId}/getNewestMessage
  • /group/{groupId}/getSubscriptions
  • /group/{groupId}/messages
  • /group/{groupId}/subscribe
  • /user/{id}/login/{gcmId}
  • /group/{groupId}/user/{userId}/acceptSubscription
  • /group/{groupId}/user/{userId}/declineSubscription

There is also a WADL document describing the REST API.
